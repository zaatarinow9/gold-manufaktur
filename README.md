# Gold Helwah Showroom

Production-ready foundation for a multilingual luxury gold jewelry catalog built with Next.js 16, the App Router, TypeScript, Tailwind CSS, and `next-intl`.

## Stack

- Next.js 16 with App Router
- TypeScript
- Tailwind CSS
- `next-intl`

## Locales

- `de` default
- `ar`
- `en`
- `fr`
- `tr`

Arabic is configured with RTL support.

## Public Routes

- `/{locale}`
- `/{locale}/shop`
- `/{locale}/contact`
- `/{locale}/impressum`
- `/{locale}/datenschutz`

The website is a showroom and catalog only. It does not include cart, checkout, or payment flows.

## Development

```bash
npm run dev
```

## Validation

```bash
npm run lint
npm run build
```
